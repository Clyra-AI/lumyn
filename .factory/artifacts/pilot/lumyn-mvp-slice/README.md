# Lumyn MVP Downstream Pilot Evidence

This directory captures the first real downstream Factory pilot proof for the
Lumyn MVP repo.

The package is intentionally honest about scope:

- T1, T2, T2.5, and T2.6 are implemented and validated.
- The full MVP is not closed.
- The first missing product slice is T3, represented by a bounded repair task.
- Live sandbox and live eval work remain blocked until credential and network
  posture are explicitly approved.

Run `python3 scripts/validate_factory_pilot_evidence.py` or
`make test-contracts` to verify the package.
